# Add board managers

## What is board?

Board -> link to template -> have multiple submissions.
Every `Submission` belongs to a `Board` and has a `Template` which is used to create the submission.

## Tasks

- [ ] Add /boards page to manager board in application.
- [ ] Every board has a template picker, which allows users to select a template for the board. (multiple)
- [ ] Every board has a button create a `Submission` with the selected template.
- [ ] Every board has a list of `Submission` created from the board, with the ability to view details of each submission.
